Pico W LED

Blink frequency

Range 1–20 Hz. Current: {{HZ}} Hz.

Reset

Restore the values from default-settings.json embedded at compile time.

HTTP API

For scripting or partial updates.

GET  /api/settings    export current settings as JSON
POST /api/settings    import partial JSON (missing fields ignored)
POST /api/reset       restore embedded defaults

Example:

curl -X POST http://192.168.4.1/api/settings \
     -H 'Content-Type: application/json' \
     -d '{"blink_hz": 5}'
